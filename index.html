Assignment 1

This assignment served to establish our online ePortfolio for the course. We first established our folder structure on the server. The page you are currently reading is on our root directory and all subsequent assignments will live in sub-directories.

The assignment first directed us to create two web pages using manually typed html. The first was this page that you're reading, and the second was the "About Me" page that will help the class get to know me better. The HTML was very basic and utilized basic tags with a few attributes thrown in for demonstration purposes. The beauty of the web is that every web page, no matter how complex, is composed of a series of relatively simple HTML tags.

What really gives life to the internet is CSS. The latter portion of the assignment had us take the two HTML pages and link them to an external style sheet, which we also created manually. The style sheet formed the basis of all fonts, colors, margins, table configurations, list configurations, and so on. CSS is extremely powerful in styling web pages down to the finest detail. It is also versatile--if I decide to completely change the look and feel of my web page at a later date, I do not need to touch the HTML code at all, but rather just the CSS. I now have a much better understanding of the relationship between HTML and CSS.

  • Establish the online ePortfolio and become familiarized with the web server structure
  • Create two simple HTML documents using a variety of tags and attributes for learning purposes
  • Create a Cascading Style Sheet (CSS) to standardize the look and feel of the current and future assignments in our ePortfolio.
  • Become familiarized with the relationship between HTML and CSS in a hands-on, purposeful way

Assignment 2

This assignment was all about JavaScript and how it interacts with HTML and CSS. We also had an introductory look and flowcharts for structured programming, which help us to visually identify the logical flow of a program.

The assignment first directed us to hone our knowledge of JavaScript, which included a detailed explanation of the DOM (Document Object Model). The DOM is the logical structure of an HTML document. Each element is a "node" and a node can have parent nodes, child nodes, and sibling nodes. Understanding this structure is critical to effectively leverage JavaScript to its maximum benefit.

We used our new knowledge of JavaScript to construct a simple program that reads inputs from an HTML form and outputs a result. In my case, I created a "pizza profiler" whereby the user answers questions about their preferences for pizza. The program takes these answers and outputs a profile based on the responses. We also utilized JavaScript to validate the program inputs. If the user enters an invalid value, JavaScript will change the corresponding HTML element to call the user's attention to it in an intuitive way. Take a look by navigating to Assignment 2!

Finally, we mapped our program out visually using a structured programming flowchart. Creating the chart was instrumental in solidifying my understanding of decision symbols, flow symbols, input/output symbols, calculation symbols, and connectors. They're not just pretty boxes anymore!

  • Learn the basics of JavaScript
  • Understand how JavaScript interacts with HTML and CSS elements and navigates an HTML document
  • Create a simple program that leverages JavaScript to validate user inputs and calculate a result based on those inputs
  • Develop a feel for structured programming flowcharts by creating a flowchart that maps the execution of the aforementioned program

Assignment 5

This was our first PHP assignment. This assignment was critical in solidifying our understanding of how client side and server side code interact. The server is able to send html to the client browser to display whatever needs to be displayed depending on the particular situation. The source code in the browser does not show any of the server side code, which is by design.

The concepts learned in this assignment are critical to understanding client/server web applications and will be critical in future assignments involving databases.

For this assignment, we implemented a dynamic address book that authenticates the user via a login.

  • Understand the fundamentals of PHP style and syntax
  • Explore the relationship between client-side and server-side code
  • Become acquainted with user authentication mechanisms
  • Explore file access, reading, and writing
